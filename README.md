# OpenList Companion v1.5.7

一款专为 Alist / OpenList 服务打造的高颜值、全功能本地桌面管理助手。将繁琐的命令行操作完全可视化，集服务一键托管、智能自动化刷新、无损数据备份、AI 智能体 MCP 接入于一体。

### 🚀 核心功能特色
* **服务一键控制中心**：支持一键启动、停止、重启 Alist 服务，运行日志实时滚动投递。内置智能端口扫描，遭遇 5244 端口占用时可一键释放。
* **智能自动任务与观影保护**：定时自动刷新或重启服务。重启时若检测到用户正在传输文件或播放流媒体（网盘视频），将自动激活保护模式顺延任务，绝不打断观影体验。
* **纯净便携、目录零污染**：程序运行时生成的本地凭证、刷新任务及 MCP 配置文件，在 Windows 系统下会自动转换为系统隐藏属性，还你清爽干净的目录。

## 📖 使用教程 (Usage Tutorial)

### 1. 📂 准备工作 (环境部署)
1. **下载解压**：在右侧的 `Releases` 区域下载最新的 `OpenListCompanion.zip` 便携包，将其解压到任意本地文件夹。
2. **导入核心**：前往 [@OpenListTeam/OpenList](https://github.com/OpenListTeam/OpenList) 下载对应的 `alist.exe`，并直接将其放置在 `OpenListCompanion.exe` 的**同级目录下**（软件会自动识别）。
   > 💡 *如果你的 `alist.exe` 放在了其他地方，也可以启动软件后点击主界面右下角的 **【选择】** 按钮手动记忆路径。*

### 2. 🚀 基础控制与面板管理
* **托管服务**：点击 **【🚀 启动服务】** 按钮，下方的运行日志会实时滚动输出 Alist 的启动日志。
* **访问网盘**：当左侧状态显示为 `🟢 正在运行` 时，点击 **【🌐 打开面板】** 按钮，会自动弹出浏览器直达本地网盘控制台。
* **账户管理**：左侧面板已经为你内置了默认账密，点击 **【复制】** 即可秒拿管理员账号 `admin`；如需修改密码，点击 **【修改管理密码】** 输入新密码后软件会自动重启生效。
* **解决冲突**：如果启动时遇到端口被占用，点击 **【🔓 释放端口】** 可以一键扫描并强行释放 `5244` 端口。

### 3. 🛡️ 自动化任务与观影保护
在“数据与自动任务”卡片中，你可以自由开关三个极具特色的便携功能：
* **【普通刷新】**：开启后，软件每隔 60 秒自动向本地 Alist 发送刷新指令，防止挂载的网盘因长时间无交互而掉线。
* **【重启刷新】**：开启后，每隔两轮刷新会自动重启一次 Alist 服务，触发全量云盘数据强制同步。
* **【播放保护】**：**（强烈推荐开启）** 当触发重启刷新时，软件会智能扫描本地活跃连接。如果你正在通过 Infuse/Plex/Netflix 等挂载端观看网盘电影或正在传输大文件，软件会**自动延期重启**，确保你的观影和传输绝对不被打断。

### 4. 🧩 AI 智能体接入 (MCP Server)
软件无缝集成了基于 [@hbestm/openlist-mcp-server](https://github.com/hbestm/openlist-mcp-server) 的 AI 生态。
1. 点击左侧导航栏的 **【🧩 MCP 设置】** 打开弹窗。
2. 在弹窗中补全你的网盘凭证后，点击 **【安装/更新 MCP】**，软件会自动调用全局 npm 脚本下载并部署最新的 MCP 环境。
3. 点击 **【启动 MCP】**，当连接地址变绿并提示监听成功时，即可实现 AI 与网盘的数据互通。
4. **客户端对接**：
   * 如果使用 **Claude Desktop**，点击 **【生成 Claude】** 即可一键完成环境写入。
   * 如果使用其他第三方 AI 客户端（如 Cline / Cursor），点击 **【生成 SOLO】** 即可导出独立配置，或直接点击 **【复制 HTTP 地址】** 进行流式 SSE 对接。
